# MCPGuard

## Source

- **arXiv**: [2510.23673](https://arxiv.org/abs/2510.23673)

## Description

Three-stage detection pipeline: static scanning → neural detector → LLM arbitrator. 96% accuracy. Identifies agent hijacking, web vulns, supply chain threats.
