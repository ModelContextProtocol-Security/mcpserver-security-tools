# Securing MCP: Tool Poisoning Defense

## Source

- **arXiv**: [2512.06556](https://arxiv.org/abs/2512.06556)
- **PDF**: https://arxiv.org/pdf/2512.06556.pdf

## Description

Layered defense framework: RSA manifest signing + LLM-on-LLM vetting + heuristic guardrails. GPT-4 blocks 71% unsafe calls. 1800 experimental runs.
