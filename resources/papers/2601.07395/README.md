# MCP-ITP: Implicit Tool Poisoning

## Source

- **arXiv**: [2601.07395](https://arxiv.org/abs/2601.07395)
- **PDF**: https://arxiv.org/pdf/2601.07395.pdf

## Description

Automated implicit tool poisoning achieving 84.2% attack success rate while keeping detection rate at 0.3%. Shows current defenses are insufficient.
