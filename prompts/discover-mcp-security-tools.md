# MCP Security Tools Research Prompt

## Goal
Discover tools, projects, and resources related to MCP (Model Context Protocol) security. We're in the discovery phase - finding what exists and understanding the landscape.

## What to search for
- MCP security scanning/auditing tools
- MCP vulnerability detection
- MCP server analyzers
- Tools that help secure MCP implementations
- Projects focused on MCP safety/security
- Related AI/LLM security tools that apply to MCP

## Output

### 1. CSV inventory
`data/mcp-security-tools.csv` with columns:
- name
- url
- description

### 2. Markdown report
`data/research-notes.md` - freeform observations including:
- Patterns you notice
- Categories or types of tools emerging
- Gaps - what doesn't seem to exist yet
- Anything surprising or notable
- Questions that came up during research
