# MintMCP Gateway

## One-liner
Enterprise MCP gateway with SOC 2 certification, centralized management, and one-click deployment for AI agents.

## Who's behind it
- **Company**: MintMCP
- **Status**: Official Cursor partner
- **Certifications**: SOC 2 compliant (monitored by Drata), AICPA SOC certified
- **Type**: Commercial MCP Gateway

## Problems it claims to solve
- Complex local deployment of MCP servers
- Difficult authentication and access control setup
- Lack of audit trails for MCP interactions
- PII/secret exposure in agent communications
- No centralized visibility across MCP servers

## Key features

### Enablement & Deployment
- STDIO server support with automatic hosting
- Central MCP registry with one-click installation
- Virtual servers with role-based access controls
- Automatic discovery and configuration

### Security & Governance
- OAuth 2.0, SAML, and SSO integration
- Complete audit logs of all MCP interactions
- Automatic PII detection and redaction
- Real-time secret scanning (API keys, credentials, tokens)

### Business Applications
- Data analysis through AI agent database queries
- Customer support with CRM/ticket system access
- Secure integration with development workflows

## How it works
1. Connect data sources via standardized MCP servers
2. Manage access centrally with security controls
3. Safely connect AI agents to governed data
